# 隐私政策

**最后更新：2026 年 4 月 30 日**

## 概述

Weread Extract（以下简称"本扩展"）尊重并保护您的隐私。本隐私政策说明本扩展如何处理用户数据。

## 数据收集

**本扩展不收集、存储或传输任何个人数据。**

具体来说：

- **不收集个人信息**：本扩展不会收集您的姓名、邮箱、IP 地址、设备信息、浏览历史或任何其他个人身份信息。
- **不传输数据**：本扩展没有任何远程服务器，所有处理均在您本地浏览器中完成。没有任何数据被发送到外部服务器。
- **不使用分析工具**：本扩展不包含任何分析、统计或跟踪代码（如 Google Analytics）。
- **不设置 Cookie**：本扩展不会设置或读取任何第三方 Cookie。

## 权限说明

本扩展申请的权限及其用途：

| 权限 | 用途 |
|------|------|
| `activeTab` | 与当前微信读书标签页通信，发送提取命令和接收内容 |
| `clipboardWrite` | 将提取的 Markdown 内容复制到剪贴板 |
| `host_permissions` (`*://weread.qq.com/*`) | 仅在微信读书阅读页面中注入提取脚本 |

## 数据存储

本扩展不存储任何用户数据。提取的内容仅在内存中临时存在，关闭弹出面板后即被释放。

## 第三方服务

本扩展不依赖、不调用任何第三方服务。

## 变更

若本隐私政策发生变更，将在本页面更新并注明最后更新日期。

## 联系我们

如有任何问题或疑虑，请通过以下方式联系：

# Privacy Policy

**Last updated: April 30, 2026**

## Overview

Weread Extract ("the Extension") respects and protects your privacy. This Privacy Policy explains how the Extension handles user data.

## Data Collection

**The Extension does NOT collect, store, or transmit any personal data.**

Specifically:

- **No personal information collected**: The Extension does not collect your name, email, IP address, device information, browsing history, or any other personally identifiable information.
- **No data transmission**: The Extension has no remote servers. All processing occurs locally in your browser. No data is sent to external servers.
- **No analytics**: The Extension does not include any analytics, statistics, or tracking code.
- **No cookies**: The Extension does not set or read any third-party cookies.

## Permissions

| Permission | Purpose |
|------------|---------|
| `activeTab` | Communicate with the active WeRead tab to send extraction commands and receive content |
| `clipboardWrite` | Copy extracted Markdown content to the clipboard |
| `host_permissions` (`*://weread.qq.com/*`) | Inject extraction scripts only on WeRead reading pages |

## Data Storage

The Extension does not store any user data. Extracted content exists temporarily in memory and is released when the popup panel is closed.

## Third-Party Services

The Extension does not use or depend on any third-party services.
